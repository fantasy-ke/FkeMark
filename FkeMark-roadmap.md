# FkeMark 功能与设计完善 · 优先级任务清单（P0 → P2）

> 排序规则：先 P0（基本功 + 防丢 + 检索），再 P1（知识化 + 发布 + 个性化），最后 P2（智能 + 生态）。
> 范围：个人笔记 / 日常写作。每项含优先级、所属分类、一句话价值说明。
> 落地前置：需提供 FkeMark 源码仓库路径（当前仅逆向了运行产物）。

---

## P0 · 第 1 阶段：基本功 + 防丢 + 检索
*目标：把"能写"变成"写得顺、不丢、找得到"*

1. **[P0] ✅ 文档内查找 / 替换** — 支持正则、大小写、选区范围；编辑器基本功，个人写作高频刚需。
2. **[P0] ✅ 跨文件全文搜索** — 遍历文件树对文件名/标题/正文建索引，结果带预览并可跳转到命中位置。
3. **[P0] ✅ 全局搜索面板（⌘P 风格）** — 统一"打开文件 / 命令 / 搜索"入口，增强现有命令菜单的过滤与最近使用。
4. **[P0] ✅ 多文档标签页 / 分屏** — 顶部文档标签 + 右键菜单（关闭 / 关闭其他 / 复制路径），支持并行记多件事。
5. **[P0] ✅ 回收站 + 软删除恢复** — 删除文件/笔记先进回收站，可还原，防误删丢失。
6. **[P0] ✅ 粘贴截图自动落盘** — 剪贴板图片自动写入文档旁 `assets/` 文件夹并用相对路径引用，告别"先保存再拖图"。
7. **[P0] ✅ 导出 PDF** — 通过浏览器打印样式或 Paged.js 精确排版，满足归档刚需（当前仅 MD/HTML/TXT）。
8. **[P0] ✅ 首启引导 Onboarding** — 首次打开的欢迎页 + 3 步引导（选文件夹→建第一篇→认识视图切换），替代纯 splash。
9. **[P0] ✅ 空状态设计** — 空白文档给写作提示 / 模板入口，而非纯空白。
10. **[P0] ✅ 设置页重组** — 左侧分组（外观/编辑器/文件/行为/快捷键/实验/关于）+ 顶部搜索，收敛当前零散项。

## P1 · 第 2 阶段：知识化 + 发布 + 个性化
*目标：从"笔记本"升级为"个人知识库"，并方便分享*

11. **[P1] ✅ 数学公式 KaTeX** — 行内 `\(...\)` 与块级 `$$...$$` 渲染；技术/学术笔记刚需。
12. **[P1] 脚注 / 参考文献** — 支持 `[^1]` 脚注与引用，长文写作必备。
13. **[P1] 片段 / 模板 Snippets** — 常用结构一键插入（日记模板、会议模板），高频复用。
14. **[P1] YAML frontmatter + 标签 `#tag`** — 文档元数据，支撑后续检索与分类。
15. **[P1] 双向链接 `[[笔记名]]` + 反向链接面板** — 把零散笔记织成网，知识库核心。
16. **[P1] 日历 / 日记视图** — 按日期浏览笔记，日记场景友好。
17. **[P1] 导出 DOCX / ePub / RTF / OPML** — 扩展发布与分享格式。
18. **[P1] 自定义导出样式（CSS 模板）+ TOC 自动生成** — 导出带目录与统一排版。
19. **[P1] 自定义主题 + CSS 注入 + 主题市场** — 突破 3 套内置主题，强化"高颜值"卖点。
20. **[P1] ✅ 快捷键自定义** — 用户可重映射命令快捷键。
21. **[P1] 可选云同步** — WebDAV / 本地文件夹同步 / GitHub，解决换机与备份。
22. **[P1] 图片管理面板** — 列出本文图片，支持重命名、替换、批量导出。
23. **[P1] 暗色模式精修 + 强调色自定义** — 提升视觉质感与辨识度。
24. **[P1] 状态栏信息丰富** — 显示字数 / 阅读时长 / 最后保存时间 / 同步状态。
25. **[P1] ✅ 图片拖拽占位 + 上传进度 + 统一 Toast 通知中心** — 用一致反馈替代散落 alert。
26. **[P1] 响应式与窗口缩放** — 窄窗口自动收起侧栏、隐藏小地图。

## P2 · 第 3 阶段：智能 + 生态
*目标：长期护城河*

27. **[P2] 拼写检查** — 中文分词 + 英文拼写，写作质量保障。
28. **[P2] AI 助手** — 续写 / 摘要 / 润色 / 翻译，可接本地或 API。
29. **[P2] 插件系统（extension 机制）** — 开放生态，第三方扩展能力。
30. **[P2] 本地版本快照 / 差异对比（diff）** — 历史回溯与改动比对。
31. **[P2] 可选图床上传** — SM.MS / 自建 / Base64 内联切换，便于外链分享。
32. **[P2] 字数目标 / 写作统计 / 阅读时长** — 写作习惯与成就激励。
33. **[P2] 关系图谱 Graph View** — 可视化双向链接网络。
34. **[P2] 演示模式** — 把笔记一键变为幻灯片演示。
35. **[P2] Emoji 选择器 / KaTeX 符号面板** — 降低插入门槛。
36. **[P2] 打字机滚动 + 段落聚焦** — 强化专注模式沉浸感。
37. **[P2] 移动端** — 跨设备随时记录。
38. **[P2] 无障碍优化** — 对比度达标、字体缩放、全键盘可达。

## 落地建议
- **节奏**：先做 P0 三件套（搜索 + 查找替换 + 回收站/标签），快速提升日常可用度，再推进 P1 知识化，避免一次堆太多导致回归。
- **关键技术点**（详见调研计划文档）：
  - 查找/替换 → Tiptap + `prosemirror-search` 或 contenteditable overlay
  - 全文搜索 → 文件树遍历 + 索引（先朴素扫描，后接 MiniSearch/FlexSearch）
  - 图片 → Tauri `fs`/`dialog` 写 `assets/`，粘贴走 `clipboard` API
  - 公式 → `katex` + Tiptap 自定义 Node
  - PDF → `window.print()` 套打印样式或 Paged.js
  - 双向链接 → 元数据索引 + 反链查询，Tauri 侧做路径解析
  - 主题/CSS → `custom.css` 注入入口 + 主题 JSON（背景/文字/强调色）
