# FkeMark 设计文档

FkeMark 是一款极简、本地优先的 Markdown 即时渲染编辑器。

## 核心理念

**文件即数据** — 不建立私有数据库，双击 .md 文件即开即写。

*极简视觉* — 扁平化、无缝窗口、冷淡现代配色。

`所见即所得` — 光标处显示源码，失焦即刻渲染。

> 好的设计是尽可能少的设计 — Dieter Rams

## 功能规划

  • - 即时渲染引擎 — 光标处显示 Markdown 标记，失焦自动隐藏
  • - 本地文件树 — 浏览和切换任意文件夹的 .md 文件
  • - 自动保存 — 300ms 防抖自动后台保存
  • - TOC 动态大纲 — 自动提取 H1-H6 标题，点击跳转
  • - 斜杠命令 — 输入 / 唤起悬浮菜单插入组件

### 技术栈

```
FkeMark / Tauri + React + TypeScript + ProseMirror
```

编辑引擎基于 ProseMirror，通过 `Decoration.widget` 动态插入语法标记。
