## 阶段一：基础排版（极简常用）


这是最基础的文本样式，决定了文档的基本骨架。


`hhhh`






### 1. 标题 (Headers)


- **语法**：使用 `#` 加上一个**空格**。`#` 的数量代表标题层级（1 到 6 级）。
- **键盘输入**：


```markdown
# 一级标题 (H1)
## 二级标题 (H2)
### 三级标题 (H3)
#### 四级标题 (H4)
##### 五级标题 (H5)
###### 六级标题 (H6)

```


- **FkeMark 快捷操作**：
- **快捷键**：`Ctrl/Cmd + 1` 到 `Ctrl/Cmd + 6` 快速切换当前行标题层级；`Ctrl/Cmd + 0` 恢复为普通正文。
- **斜杠命令**：空行输入 `/h1`、`/h2` 或 `/h3` 后回车。


### 2. 粗体、斜体、删除线 (Emphasis)


- **语法**：用特定符号包裹需要修饰的文字。
- **键盘输入**：


```markdown
这是 *斜体* 文本（也可以用 _斜体_）
这是 **粗体** 文本（也可以用 __粗体__）
这是 ***粗斜体*** 文本
这是 ~~删除线~~ 文本

```


- **FkeMark 快捷操作**：
- **快捷键**：选中文字后，按 `Ctrl/Cmd + B`（粗体）、`Ctrl/Cmd + I`（斜体）。
- **即时渲染行为**：光标在文字内时显示 `粗体`，光标移开后，`` 隐形，文字加粗。


### 3. 无序列表与有序列表 (Lists)


- **语法**：使用 `-`、`*`、`+` 或数字加 `.`，后跟一个**空格**。
- **键盘输入**：


```markdown
- 无序列表项 A
- 无序列表项 B
  - 按 Tab 键可以缩进，产生子列表

1. 有序列表项 一
2. 有序列表项 二

```


- **FkeMark 快捷操作**：
- **回车智能延续**：在列表项末尾敲击回车（Enter），下一行会自动生成 `- `或 `3. `。连续敲击两次回车可退出列表。
- **缩进调整**：按 `Tab` 增加缩进（降级），按 `Shift + Tab` 减少缩进（升级）。

## 阶段二：结构元素（组织内容）


用于将零散的文字重组为逻辑清晰的结构。


### 1. 引用块 (Blockquotes)


- **语法**：使用 `>` 加上一个**空格**。
- **键盘输入**：


```markdown
> 这是一段引用文本。
> 引用内可以换行，也可以嵌套。
>> 这是一个嵌套的二级引用。

```


- **FkeMark 快捷操作**：
- **快捷键**：`Ctrl/Cmd + Shift + Q`。
- **斜杠命令**：输入 `/quote` 回车。


### 2. 超链接与图片 (Links & Images)


- **语法**：使用 `显示文本` 表示链接；在前面加 `!` 表示图片。
- **键盘输入**：


```markdown
[访问百度](https://www.baidu.com)

![图片替代文字](https://example.com/image.png "悬停提示文字")

```


- **FkeMark 快捷操作**：
- **快捷键**：选中文字后按 `Ctrl/Cmd + K`，直接弹出链接输入框。
- **拖拽智能识别**：支持从本地直接拖拽图片文件到编辑器中。FkeMark 会通过 Tauri 后台将图片安全复制到文档同级的 `assets` 目录下，并自动转化为本地相对路径：。

### 3. 分割线 (Horizontal Rules)


- **语法**：在一行中使用三个或更多的 `*`、`-` 或 `_`。
- **键盘输入**：


```markdown
***
---
___

```


- **FkeMark 快捷操作**：输入 `---` 并按回车，编辑器会自动将其渲染为一条横跨整行的淡灰色视觉分割线。

## 阶段三：高级排版（效率工具）


主要针对程序员、学者、项目经理等专业用户的排版工具。


### 1. 任务列表 (Task Lists / To-Do)


- **语法**：在无序列表的基础上，加上 `[ ]`（未完成）或 `[x]`（已完成）。
- **键盘输入**：


```markdown
- [ ] 待办任务 A（注意中括号内有一个空格）
- [x] 已完成任务 B

```


- **FkeMark 快捷操作**：
- 支持用鼠标直接点击选框，前端会自动修改底层的 Markdown 文本（将 `[ ]` 替换为 `[x]`）。
- **斜杠命令**：输入 `/todo` 回车。


### 2. 行内代码与代码块 (Code)


- **语法**：单个反引号 ``` 包裹行内代码；三个反引号 ````` 包裹多行代码块，并可指定语言。
- **键盘输入**：


```markdown
请在终端运行 `npm run dev` 命令。

```


const message = "Hello, FkeMark!"; console.log(message);


```plaintext


```


```plaintext


* **FkeMark 快捷操作**：
* **代码高亮**：输入 ````javascript` 并回车，编辑器会生成带语法高亮的编辑框，并在右下角提供语言选择下拉菜单。
* **斜杠命令**：输入 `/code` 快速插入代码块。



### 3. 表格 (Tables)

* **语法**：使用 `|` 划分列，使用 `-` 划分表头与内容。在分隔线中加入 `:` 可控制对齐方式。
* **键盘输入**：
```


|  |
|  |
|  |
|  |


```plaintext


* **FkeMark 快捷操作**：
* **拖拽/自动延续**：在单元格中按 `Tab` 自动跳转到下一个单元格，如果在最后一行的最后一格按 `Tab`，会自动新建一行。
* **快捷新建**：输入 `/table` 弹出 `3x3` 的网格滑块，鼠标拖拽选择行列数后一键生成表格。

## 阶段四：复杂语法（扩展与公式）

支持科技论文与复杂文档排版的扩展语法。

### 1. 数学公式 (LaTeX Math)

* **语法**：单美刀符号 `$` 包裹行内公式；双美刀符号 `$$` 包裹独占一行的块级公式。
* **键盘输入**：
* 行内公式：
```


勾股定理公式为 $a^2 + b^2 = c^2$。


```plaintext


* 块级公式：
```


$$ f(x) = \int_{-\infty}^{\infty} e^{-x^2} dx $$


```plaintext




* **FkeMark 智能渲染**：
* 公式输入完毕后，光标离开即刻通过 **KaTeX** 渲染出印刷级的数学公式。
* 当光标重新点入公式内部时，自动展开为 LaTeX 源码。



### 2. 脚注 (Footnotes)

* **语法**：使用 `[^1]` 创建标注，在文档底部使用 `[^1]: 详细解释` 写入具体内容。
* **键盘输入**：
```


这是一个需要解释的名词[^1]。





[^1]: 这里是该名词的详细脚注解释内容。


```plaintext



### 3. 字符转义 (Escaping)

* **语法**：如果你只想显示 Markdown 的控制符号（如只想输入 `*` 而不想让文字变斜体），在符号前加反斜杠 `\`。
* **键盘输入**：
```


\*这段文字两侧会带有真正的星号\* \\ 这是一个反斜线


```plaintext
