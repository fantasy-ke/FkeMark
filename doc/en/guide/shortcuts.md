# Shortcuts

FkeMark shortcuts are split into **app shortcuts** and **editor shortcuts**. App shortcuts control files, modes, and panels; editor shortcuts only apply inside the document editor.

## App shortcuts

| Shortcut | Action |
| --- | --- |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | New file |
| <kbd>Ctrl</kbd> + <kbd>O</kbd> | Open file |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>O</kbd> | Open folder |
| <kbd>Ctrl</kbd> + <kbd>S</kbd> | Save current document |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> | Cycle editor mode |
| <kbd>F11</kbd> | Focus mode |
| <kbd>Ctrl</kbd> + <kbd>F</kbd> | Find |
| <kbd>Ctrl</kbd> + <kbd>H</kbd> | Replace |
| <kbd>Ctrl</kbd> + <kbd>P</kbd> | Command palette |
| <kbd>Ctrl</kbd> + <kbd>W</kbd> | Close current tab |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | Recycle bin |

## Editor shortcuts

| Shortcut | Action |
| --- | --- |
| <kbd>Ctrl</kbd> + <kbd>1</kbd> ~ <kbd>6</kbd> | Toggle heading level 1 to 6 |
| <kbd>Ctrl</kbd> + <kbd>0</kbd> | Paragraph |
| <kbd>Ctrl</kbd> + <kbd>B</kbd> | Bold |
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | Italic |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> | Strike |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Q</kbd> | Blockquote |
| <kbd>Ctrl</kbd> + <kbd>K</kbd> | Insert or edit link |
| <kbd>Tab</kbd> | Move to the next table cell; create a new row from the last cell |
| <kbd>Enter</kbd> | Convert a standalone <code>---</code> line to a divider; create a code block after <code>&#96;&#96;&#96;lang</code> |

## Presentation mode navigation

| Shortcut | Action |
| --- | --- |
| <kbd>←</kbd> / <kbd>PageUp</kbd> | Previous slide |
| <kbd>→</kbd> / <kbd>Space</kbd> / <kbd>PageDown</kbd> | Next slide |
| <kbd>Home</kbd> | First slide |
| <kbd>End</kbd> | Last slide |
| <kbd>Esc</kbd> | Exit presentation or close overlays |

## Custom shortcuts

Open the “Shortcuts” section in [Settings](./settings) to change combinations or reset them to defaults.

::: tip
The documented <kbd>Ctrl</kbd> key also matches <kbd>Command</kbd> on macOS. The actual active mapping is shown in the app settings page.
:::
