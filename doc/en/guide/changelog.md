# Changelog

FkeMark’s official release notes are maintained in GitHub Releases. The docs header includes a “Versions” dropdown that links to release history and this changelog entry.

## Quick links

- [Release history](https://github.com/fantasy-ke/FkeMark/releases): view all released versions and installers.
- [Latest release](https://github.com/fantasy-ke/FkeMark/releases/latest): open the current latest release page.

## How to use these links

| Entry | Best for |
| --- | --- |
| Release history | Compare versions or download older installers |
| Latest release | Download the current stable build and read the newest notes |
| Docs guide | Learn usage, deployment, and theme configuration |
