# 快速开始

FkeMark 是一款桌面 Markdown 编辑器，目标是：**让文件仍然是普通 `.md`，编辑体验更接近成稿**。

## 三分钟工作流

1. 打开 FkeMark。
2. 选择一个本地文件夹作为文档目录。
3. 新建或打开 `.md` 文件。
4. 在 `Live`、`Read`、`Source` 三种模式中切换。
5. 需要对照时开启分屏；需要沉浸时开启专注模式或打字机模式。

::: tip 为什么强调本地文件？
你的笔记不依赖数据库，也不依赖某个账号。只要能读写 Markdown，其他编辑器、Git、同步盘都能继续使用这些文件。
:::

## 核心概念

| 概念 | 作用 | 适合场景 |
| --- | --- | --- |
| Live | 混合即时渲染，边写边接近预览 | 日常写作、教程整理 |
| Read | 只读预览，专注阅读成稿 | 校对、演示、检查排版 |
| Source | 纯 Markdown 源码 | 精确控制语法、批量修改 |
| 分屏 | 源码与预览并排 | 长文、表格、代码块检查 |
