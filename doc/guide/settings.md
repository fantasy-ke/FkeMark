# 设置页面

设置页面用于统一调整外观、编辑体验、视图、行为、图片、语言、快捷键、AI 辅助和实验选项。

![FkeMark 设置页面](/images/guide/settings.svg)

## 设置分区

| 分区 | 可调整内容 | 推荐使用场景 |
| --- | --- | --- |
| 外观 | 主题、字体、字号、行高、圆角 | 让阅读和写作更舒服 |
| 编辑器 | 默认编辑模式、标记显示、自动括号、拼写检查 | 调整日常输入体验 |
| 视图 | 编辑宽度、小地图、专注模式、工具栏位置 | 适配长文、宽屏或小屏 |
| 行为 | 自动保存、关闭窗口动作 | 避免误关或丢失内容 |
| 图片 | 图片保存、上传或资源处理相关选项 | 统一 Markdown 图片管理方式 |
| 语言 | 中文 / English | 切换界面语言 |
| 快捷键 | 应用和编辑器快捷键 | 建立自己的高频操作习惯 |
| AI 辅助 | 本地或 OpenAI-compatible API 配置 | 续写、润色、摘要、翻译等辅助写作 |
| 实验选项 | Mermaid、Vim 等增强能力 | 按需开启，不影响基础写作 |
| 关于 | 版本、更新通道、开发工具入口 | 检查版本或更新 |

## 建议配置

### 长文写作

- 编辑宽度：中等或宽。
- 专注模式：开启。
- 小地图：长文或代码较多时开启。
- 自动保存：保持开启。

### 项目文档

- 默认模式：Live。
- 显示 Markdown 标记：开启，便于检查语法。
- 工具栏：固定在顶部或靠近编辑区。
- 快捷键：保留保存、查找、链接、标题等高频项。

### 演示和校对

- 使用 Read 模式检查成稿效果。
- 使用演示模式前，用独立 <code>---</code> 分隔每页幻灯片。
- 需要中英文混写时，保持拼写检查开启。

## 常见问题

### 设置会影响 Markdown 文件内容吗？

不会。多数设置只影响 FkeMark 的显示和编辑体验，不会改写你的 Markdown 文件。

### 快捷键改错了怎么办？

到“快捷键”分区点击单项恢复，或使用“全部恢复默认”。

### AI 辅助是否必须配置？

不是。AI 辅助默认关闭；不开启时不影响本地 Markdown 编辑流程。
