# 更新日志

FkeMark 的正式发布记录以 GitHub Releases 为准。文档站在头部提供“版本”下拉，用来跳转历史版本和当前更新日志入口。

## 快速入口

- [历史版本](https://github.com/fantasy-ke/FkeMark/releases)：查看所有已发布版本与安装包。
- [最新发行版](https://github.com/fantasy-ke/FkeMark/releases/latest)：直接进入当前最新版本页面。

## 阅读建议

| 入口 | 适合场景 |
| --- | --- |
| 历史版本 | 对比不同版本、下载旧版本安装包 |
| 最新发行版 | 下载当前稳定包、查看最新变更说明 |
| 文档教程 | 了解功能用法、部署方式和主题配置 |
