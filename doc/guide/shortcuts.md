# 快捷键

FkeMark 的快捷键分为 **应用快捷键** 和 **编辑器快捷键**。应用快捷键控制文件、模式和面板；编辑器快捷键只在正文编辑区内生效。

## 应用快捷键

| 快捷键 | 动作 |
| --- | --- |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | 新建文件 |
| <kbd>Ctrl</kbd> + <kbd>O</kbd> | 打开文件 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>O</kbd> | 打开文件夹 |
| <kbd>Ctrl</kbd> + <kbd>S</kbd> | 保存当前文档 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> | 切换编辑模式 |
| <kbd>F11</kbd> | 专注模式 |
| <kbd>Ctrl</kbd> + <kbd>F</kbd> | 查找 |
| <kbd>Ctrl</kbd> + <kbd>H</kbd> | 替换 |
| <kbd>Ctrl</kbd> + <kbd>P</kbd> | 命令面板 |
| <kbd>Ctrl</kbd> + <kbd>W</kbd> | 关闭当前标签 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd> | 回收站 |

## 编辑器快捷键

| 快捷键 | 动作 |
| --- | --- |
| <kbd>Ctrl</kbd> + <kbd>1</kbd> ~ <kbd>6</kbd> | 切换一级到六级标题 |
| <kbd>Ctrl</kbd> + <kbd>0</kbd> | 切换为普通段落 |
| <kbd>Ctrl</kbd> + <kbd>B</kbd> | 加粗 |
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | 斜体 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd> | 删除线 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Q</kbd> | 引用块 |
| <kbd>Ctrl</kbd> + <kbd>K</kbd> | 插入或编辑链接 |
| <kbd>Tab</kbd> | 在表格单元格中跳到下一格，最后一格会新建行 |
| <kbd>Enter</kbd> | 在独立 <code>---</code> 行生成分割线；在 <code>&#96;&#96;&#96;lang</code> 后生成代码块 |

## 演示模式导航

| 快捷键 | 动作 |
| --- | --- |
| <kbd>←</kbd> / <kbd>PageUp</kbd> | 上一页 |
| <kbd>→</kbd> / <kbd>Space</kbd> / <kbd>PageDown</kbd> | 下一页 |
| <kbd>Home</kbd> | 第一页 |
| <kbd>End</kbd> | 最后一页 |
| <kbd>Esc</kbd> | 退出演示或关闭弹层 |

## 自定义快捷键

打开 [设置页面](./settings) 的“快捷键”分区，可以逐项修改组合键，也可以一键恢复默认值。

::: tip
文档中的 <kbd>Ctrl</kbd> 在 macOS 上也会匹配 <kbd>Command</kbd>，实际显示仍以应用内设置页为准。
:::
