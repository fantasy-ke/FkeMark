# 功能模块与按钮

本页汇总近期补充到编辑器的功能模块和入口按钮。按钮是否直接显示，取决于“设置页面 → 外观 → 工具栏布局”；隐藏按钮可以拖回工具栏，也可以放入格式、块级、插入或工具分组。

## 编辑器功能模块

| 模块 | 入口 | 主要用途 | 使用提示 |
| --- | --- | --- | --- |
| 本地版本历史 | 工具栏“版本历史”按钮 | 保存前自动留存快照，也可以手动“创建当前快照”，用于比较和恢复。 | 需要先保存当前文档；恢复历史版本后，编辑器内容会被替换并保持未保存状态。 |
| 版本差异对比 | 版本历史中的对比视图 | 查看历史快照到当前内容的新增、删除和上下文。 | 长文对比时使用“上一个差异”“下一个差异”按钮跳转，不需要只靠滚动查找。 |
| 片段与模板 | 工具栏“打开片段与模板”按钮 | 插入日记、会议记录、待办、技术笔记、读书笔记等内置模板，或维护自己的 Markdown 片段。 | 自定义片段可使用日期和时间变量，插入时会替换为当前值。 |
| 写作质量检查 | 工具栏“拼写检查”按钮 | 显示中英文统计，并检查中文错词、常见英文错拼、重复词和重复标点。 | 需要在设置中启用拼写检查；英文系统词典仍可通过右键查看建议。 |
| 演示模式 | 工具栏“演示模式”按钮 | 将当前 Markdown 按幻灯片方式阅读或展示。 | 单独一行的 <code>---</code> 会开始新幻灯片；方向键、空格或翻页键切换，<kbd>Esc</kbd> 退出。 |

## 工具栏按钮清单

| 按钮 | 作用 | 备注 |
| --- | --- | --- |
| H 标题下拉 | 切换 H1-H6 或正文段落 | 适合快速调整文档层级。 |
| 加粗 / 斜体 / 删除线 / 行内代码 | 插入或切换行内格式 | 支持选中文本后直接套用。 |
| 引用 / 无序列表 / 有序列表 / 任务列表 / 分割线 | 插入常见块级结构 | 有序列表按钮带样式下拉，可选择数字、字母或罗马数字。 |
| 表格 | 通过网格选择行列并插入表格 | 适合快速创建小型结构化信息。 |
| 链接 / 双链 / 图片 | 插入外部链接、Wiki 双链或图片语法 | 图片仍建议放在当前文档同级资源目录。 |
| 代码块 / 斜杠命令 | 插入代码块或打开命令菜单 | 代码块可继续选择语言。 |
| 版本历史 | 查看、创建、对比和恢复本地快照 | 差异窗口提供上/下一个差异导航。 |
| 片段与模板 | 插入内置模板或个人片段 | 适合重复结构，如会议记录、读书笔记。 |
| 拼写检查 | 打开写作质量检查面板 | 受“设置 → 编辑器 → 拼写检查”开关控制。 |
| 演示模式 | 将当前文档作为幻灯片展示 | 单独一行 <code>---</code> 分页。 |

## 相关设置

| 设置 | 路径 | 说明 |
| --- | --- | --- |
| 工具栏布局 | 设置页面 → 外观 → 工具栏布局 | 拖拽按钮调整顺序、隐藏入口、移动分隔线，或把按钮收进分组。 |
| 显示行号 | 设置页面 → 视图 → 显示行号 | 在编辑器左侧显示行号，长文实时编辑时行号会继续跟随内容增长。 |
| 版本快照保留数量 | 设置页面 → 行为 → 版本快照保留数量 | 每个文件可选择最多保留 10、25、50 或 100 次本地版本快照。 |

## 推荐使用方式

- 日常写作：保留标题、列表、链接、图片、表格和版本历史按钮。
- 长文校对：开启显示行号、小地图和版本差异导航，方便定位改动。
- 重复格式：把会议记录、技术笔记、读书笔记等结构保存为片段。
- 演示材料：用 <code>---</code> 分页，保存后通过演示模式检查每页节奏。
