# 工具栏

工具栏把常用写作动作集中在编辑区附近，适合不想记语法时快速插入 Markdown 结构。

![FkeMark 工具栏区域](/images/guide/toolbar.svg)

## 常用区域

| 区域 | 常见动作 | 适合场景 |
| --- | --- | --- |
| 文件 | 新建、打开、保存 | 开始写作或切换文档 |
| 模式 | Live、Read、Source、分屏 | 在编辑、阅读和源码控制之间切换 |
| 格式 | 标题、加粗、斜体、删除线、引用、链接 | 处理段落和行内格式 |
| 插入 | 图片、表格、代码块、分割线 | 插入结构化内容 |
| 工具 | 写作质量、AI 辅助、演示模式 | 校对、生成辅助和展示 |

## 推荐用法

- **起草阶段**：常用标题、列表、引用、代码块。
- **整理阶段**：补充链接、图片、表格和分割线。
- **校对阶段**：打开写作质量检查，切换到 Read 或分屏检查排版。
- **演示阶段**：用演示模式把当前 Markdown 按分隔线切成幻灯片。

## 浮动工具栏

默认工具栏更靠近编辑区域，减少鼠标移动距离。若你更喜欢固定布局，可以到 [设置页面](./settings) 调整工具栏位置和显示方式。

::: tip
工具栏是快捷入口，不会替代 Markdown 语法。熟悉语法后，可以直接输入 Markdown，再用工具栏处理图片、表格、链接等高频操作。
:::
